My Projects

ChatNode

A social media platform for contributions

ChatNode is a unique, open-source social media platform that operates on a unique principle: users must contribute in order to access others' contributions. This platform allows users to post a range of materials, from projects to research work, courses, and other valid documents. Upon joining, users receive a 10 credit-point bonus, which they can use to access other users' work. However, each access costs 2 point, which is debited from the user's account, while the user whose work has been accessed is credited with 1 point. This system ensures that every user is incentivized to contribute to the platform, creating a vibrant community of knowledge-sharing and collaboration. In short, ChatNode is a platform where the only condition for participation is active contribution, allowing for a dynamic and mutually-beneficial community of learners and creators.
